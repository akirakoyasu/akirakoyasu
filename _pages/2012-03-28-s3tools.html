Usage

バイナリパッケージを解凍してできたディレクトリ内にあるbin/に$PATHを通してください。

更に、環境変数 $S3TOOLS_ACCESSKEY_IDにAWSのアクセスキーID、$S3TOOLS_SECRET_ACCESSKEYにAWSのシークレットアクセスキーを設定しておきます。

$ s3-list s3://backetName/

$ s3-get s3://backetName/key

$ s3-put file s3://backetName/key

Build

ソースパッケージはMavenプロジェクトになっています。パッケージを解凍してできたディレクトリへ移動して、

$ mvn package

でビルドできます。
