Usage

バイナリパッケージを解凍してできたディレクトリ内にあるbin/に$PATHを通してください。

更に、環境変数 $R53TOOLS_ACCESSKEY_IDにAWSのアクセスキーID、$R53TOOLS_SECRET_ACCESSKEYにAWSのシークレットアクセスキーを設定しておきます。

$ r53-list-zone

$ r53-create-zone domain.com

$ r53-list-record domain.com.

$ r53-create -d domain.com. -n www.domain.com -t A -l 86400 111.222.333.444

$ r53-delete -d domain.com. -n www.domain.com -t A -l 86400 111.222.333.444

Build

ソースパッケージはMavenプロジェクトになっています。パッケージを解凍してできたディレクトリへ移動して、

$ mvn package

でビルドできます。
